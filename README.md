About
===

**DGSerial** is **Serial Port** helper library for *.NET.

## Download

+ **[NuGet](https://www.nuget.org/packages/DG.Serial)**
+ **[Latest Release](../../releases/latest)** 

## Requirements

* Microsoft Windows with .NET Framework 4, .NET Standard 2 or later
